Funktionalanalysis

Funktion Erklärung
L(f,x) Laplace-Transformation von f an der Stelle x.
Beispiel: L(f,x); f(t):=t.
Die Berechnung führt in bestimmten Bereichen zu einem großen numerischen Fehler. Bitte mit dem symbolischen Ergebnis vergleichen.
delta(x,a) Nadelimpuls der Höhe a. Flächeninhalt unter dem Graphen bleibt immer gleich eins.

Hilfsfunktionen

Funktion Erklärung
rand() Zufallszahl aus dem Intervall [0,1].
rand(a,b) Zufallszahl aus dem Intervall [a,b].
rand(a) Wählt zufällig ein Element aus der Liste a.
rand(a:b) Wählt zufällig ein Element aus [a,a+1,a+2,...,b].
rand(a:b:d) Wählt zufällig ein Element aus [a,a+d,a+2d,...,b].
af(x0,y0,x1,y1) Die affine Funktion durch (x0, y0) und (x1, y1).
Beispiel: f:=af(2,1,6,2),f(x).
tg(f,a,x) Tangente: f(a)+f'(a)*(x-a).
sc(f,a,b,x) Sekante: f(a)+(f(b)-f(a))/(b-a)*(x-a).
clamp(x,a,b) Bedeutungsgleich mit min(max(x,a),b).
roots(f) Versucht, die Nullstellen von f(x) für x∈[−100,100] zu bestimmen.
roots(f,a,b) Versucht, die Nullstellen von f(x) für x∈[a,b] zu bestimmen.
map(f,a) Wendet f auf jedes Element der Liste a an.
Beispiel: map(x.2x,1:10).
filter(p,a) Die Elemente aus a, die das Prädikat p erfüllen.
Beispiel: filter(k.mod(k,2)=1,1:10).

Eingabe-Beispiele

sin(x)
Eine explizite Funktion.
sin(x);scale(0.1),P(pi,0)
Zeichne eine Funktion an der Position P(x,y) mit der Skalierung scale(dx,dy=dx).
sin(x);;scale(0.1),P(pi,0)
Alles hinter zwei Semikolons wird nur einmal ausgeführt und dann entfernt.
sin(x),cos(x),tan(x),cot(x)
Viele Funktionen.
x^2-x^2=1
Eine implizite Funktion.
x^2-x^2=1,x,-x
Eine implizite Funktion und ihre Asymptoten.
f(x):=ln(x),f(x),f'(x)
Eine Funktion und ihre Ableitung.
f(x):=sin(x),a:=2,f(x),f(a)+f'(a)*(x-a)
Tangente an eine Funktion.
exp(x),sum(0,4,k.x^k/fac(k))
Partialsumme der Exponentialreihe.
int(0,10,t.t^(x-1)*exp(-t))
Gamma-Funktion via Parameterintegral.
[cos(t),sin(2t)/2]
Eine Parameterkurve.
t0:=0,t1:=10pi,[t/10*cos(t),t/10*sin(t)]
Eine Parameterkurve für den angegebenen Definitionsbereich.
r(t):=sin(5t)+4,[r(t)*cos(t),r(t)*sin(t)]
Ein Stern.
cos(sqrt(x^2+y^2))
Eine bivariate Funktion.
[cos(u)sin(v),sin(u)sin(v),cos(v)]
Eine parametrische Fläche.
[(R+r*cos(v))cos(u),(R+r*cos(v))sin(u),r*sin(v)];
R:=4,r:=2,v1:=2pi
Ein parametrischer Torus.
